# Crypto Tracker

A **Streamlit** application to track and visualize historical cryptocurrency data for Bitcoin and Ethereum. The app allows users to customize their view by selecting date ranges, data intervals, and specific metrics like Open, High, Low, Close, or Volume.

---

## Features

- **Cryptocurrency Selection**: Choose between Bitcoin (`BTC-USD`) and Ethereum (`ETH-USD`).
- **Custom Date Range**: Specify the start and end dates for the historical data.
- **Flexible Data Intervals**: Options range from 1 minute to 3 months.
- **Metric Visualization**: Select specific metrics such as Open, High, Low, Close, or Volume for visualization.
- **Interactive Charts**: Powered by **Plotly**, providing an interactive and visually appealing experience.

---

## Technology Stack

- **Python**: Core programming language.
- **Streamlit**: Framework for building interactive web applications.
- **yFinance**: Fetching historical cryptocurrency data from Yahoo Finance.
- **Plotly**: For creating interactive and dynamic visualizations.
- **Dateutil**: For relative date calculations.

---

## Installation and Setup

### Prerequisites
- Python 3.8 or later installed on your system.
- Required Python libraries:
  - `yfinance`
  - `streamlit`
  - `pandas`
  - `plotly`
  - `python-dateutil`
